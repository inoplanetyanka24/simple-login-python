# Simple Python Login Script
This repository contains a basic Python script that demonstrates a simple login system.
The program checks a username and password entered by the user against stored credentials and prints feedback.

## Features
- Prompts the user for a username and password.
- Validates input against stored values.
- Provides clear feedback:
  - "Welcome!" if both are correct.
  - "Password is wrong." if the username is correct but the password is wrong.
  - "Username is wrong." if the username does not match.
